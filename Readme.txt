The compared algorithms involved in the manuscript named "MREKLM: A fast multiple random empirical kernel learning machine".

Directory:
0 Datasets 	-> 	The datasets directory
1 MREKLM	->	The proposed Algorithm MREKLM
2 MEKL		->	The traditional MEKL algorithm
3 GMKL		-> 	More Generality in Efficient Multiple Kernel Learning
4 NLMKL		->	The algorithm of Non-Linear Combinations of Kernels
5 RBMKL		->	The algorithm of Rule-based Multiple Kernel Learning
6 SVM		-> 	The Support vecgtor machines with RBF kernel
7 MSE		-> 	The linear MSE algorithm
